Petit manuel d'installation

Pour Windows

Téléchargez la dernière version de flutter pour Windows ici.

Extraire les fichiers et placez le dossier Flutter à l'emplacement choisi (par ex: C:\src\flutter; ne pas installer Flutter dans un répertoire qui nécessite des privilèges.)

Mettre à jour le PATH

Si vous désirez éxécuter les commandes de Flutter dans le terminal vous devez mettre jour le PATH. Pour ce faire rendez vous dans votre barre de recherche Windows, tapez "env" et choisissez "Edit environment variables for your account".

En dessous de User Variables vous devriez trouver une entrée nommé PATH. Si elle existe vous devez rajouter flutter/bin avec ; pour séparer les valeurs. Si elle n'existe pas créez une nouvelle variable nommé PATH avec le chemin complet jusque flutter/bin

Il est à noter que vous devez fermer et réouvrir le terminal pour que le chemin se mette à jour.

Run flutter doctor

Nous allons maintenant pouvoir ouvrir une console dans le répertoire de flutter et taper flutter doctor

Cette commande sert à vérifier votre environement et retourne le statut de votre installation de Flutter. Vous devriez voir quelque chose comme ceci :

[-] Android toolchain - develop for Android devices • Android SDK at D:\Android\sdk ✗ Android SDK is missing command line tools; download from https://goo.gl/XxQghQ • Try re-installing or updating your Android SDK, visit https://flutter.dev/setup/#android-setup for detailed instructions.

Comme vous pouvez le voir ici on peut remarquer qu'Android Studio SDK est manquant (vous trouverez tout les éléments manquant à l'installation correcte de Flutter, et nous allons les expliquer les uns après les autres).

Installez Android Studio

Pour commencer vous pouvez télécharger la dernière version d'Android Studio ici.

Lancez Android Studio et aller sur "Android Studi Setup Wizard", cela va installer les dernières versions d'Android SDK, Android SDK Platform-Tools et Android SDK Build-Tools qui sont requis pour les applications Flutter.

Mise en place de votre Android Device

Pour préparer et tester votre application Flutter, vous avez besoin d'un appareil avec Android 4.1 (ou plus)

  1. Activez "Developper options" et "USB Debugging" si votre appareil. Pour plus d'information vous pouvez lire la documentation Android.
  2. Installez Google USB Driver.
  3. Branchez votre téléphone par USB à votre ordinateur. Si votre téléphone le demande, autorisé l'ordinateur à accéder à votre appareil.
  4. Dans le terminal, tapez flutter devices pour vérifier que votre appareil est bien reconnu par Flutter.

Installez Android Emulator

Pour préparer et tester votre application Flutter, vous pouvez passer par un émulateur de device.

  1. Activez VM acceleration.
  2. Lancez Android Studio > Tools > Android > AVD Manager et selectionnez Create virtual device.
  3. Choisissez votre device et cliquez sur Next.
  4. Selectionnez une ou plusieurs images pour la version d'Android que vous souhaitez. Une image x86 ou x86_64 est recommendée.
  5. En dessous de Emulated performance, sélectionnez Hardware - GLES 2.0 pour activer l'acceleration d'hardware.
  6. Vérifiez la configuration AVD et cliquez sur Finish
  7. Dans Android Virtual Device Manager, cliquez sur Run. L'émulateur se lancera et affichera le canevas par défaut de l'OS choisie.

Installez les plugins Dart et Flutter

Maintenant que tout a été installé, nous allons devoir installer les plugins Dart et Flutter sur notre IDE.

Pour Android Studio

  1. Lancez Android Studio.
  2. Ouvrez File > Settings > Plugins.
  3. Selecionnez Marketplace et installez le plugin Flutter.
  4. Cliquez sur YES pour installer aussi le plugin Dart.
  5. Cliquez sur RESTART pour relancer votre IDE.

Pour Visual Studio Code

  1. Lancez VS Code.
  2. Appuyez sur CTRL + Maj + P ou cliquez sur View > Command Palette.
  3. Tapez Install puis choisissez Extensions: Install Extensions.
  4. Tapez Flutter et installez le plugin. Le plugin Dart s'installera automatiquement.

Bravo ! Vous êtes arrivés à la fin de l'installation de flutter ! La prochaine étape sera la création de votre première application que nous verrons ensemble !

Pour ceux qui auraient rencontré des problèmes dans l'installation de Flutter, de l'émulateur (ou si comme moi votre ordinateur a décidé de s'envoler à chaque essai), vous pouvez directement coder l'application ici.

Pour Mac/OS

Téléchargez la dernière version de flutter pour Mac/OS ici.

Extraire les fichiers et placez le dossier Flutter à l'emplacement choisi.

Mettre à jour le PATH

Si vous désirez éxécuter les commandes de Flutter dans le terminal vous devez mettre jour le PATH. Pour ce faire rendez vous dans votre terminal, tapez $HOME/.bash_profile ou $HOME/.bashrc selon votre version de Mac/OS et mettez à jour le chemin comme ceci :

export PATH="$PATH:[PATH_TO_FLUTTER_GIT_DIRECTORY]/flutter/bin"

Pour vérifier si le chemin à bien été modifié tapez echo $PATH

Il est à noter que vous devez fermer et réouvrir le terminal pour que le chemin se mette à jour.

Installez Xcode

Installez la dernière version de Xcode depuis votre Mac App Store ou ici.

Configurez XCode avec la ligne de commande suivante :

sudo xcode-select --switch /Applications/Xcode.app/Contents/Developer
sudo xcodebuild -runFirstLaunch

Vérifiez si vous avez les licenses signées sinon tapez sudo xcodebuild -license

Créer votre application

Pour créer une nouvelle application, tapez flutter create my_app

Quand celle-ci est installée, rentrez dans le répertoire en tapant cd my_app

Pour lancer votre app tapez flutter run

Mise en place de le simulateur IOS

Sur votre Mac, trouvez Simulator via le Spotlight ou tapez la commande open -a Simulator

Assurez-vous que le simulteur utilise la version 64-bit en vérifiant dans le menu Hardware > Device.

Sur certains appareil, l'application peut déborder selon la taille de l'écran. Vous pouvez changer ça en allant dans le menu Window > Scale

Déployer sur un appareil iOS

Installez CocoaPods en tapant la commande sudo gem install cocoapods suivi de pod setup

Suivez les instructions de XCode

  1. Ouvrez le dossier par défaut dans XCode en tapant open ios/Runner.xcworkspace dans le terminal depuis le répertoir de l'application Flutter.
  2. Sélectionnez l'appareil que vous voulez déployer dans le menu déroulant.
  3. Cliquez sur Runner dans la barre de navigation latérale
  4. Vérifiez que le Development Team est activé. Pour ce faire aller dans General > Signing > Team pour Xcode 10. Dans Signing & Capabilities > Team pour Xcode 11.Lors de votre premier projet, il vous faudra entrer votre ID Apple.
  5. La première fois que vous rattacherez votre appareil à votre ordinateur, il faudra lui faire confiance en cliquant sur Trust. Pour ce faire allez dans les options de l'appareil dans General > Device Management
  6. Si la connexion automatique a échoué, vérifiez que la valeur dans General > Identity > Bundle Identifier est unique.
  7. Maintenant tapez flutter run pour lancer votre application.
